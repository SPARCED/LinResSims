By default, simulation results are saved in this folder.

Generated output files will have the following folder structure:


output   
├── sim_name   
│   ├── drs_<drug_name>   
│   │   ├── drs_<drug_name>_<replicate_id>    
│   │   │   ├── <drug_name>_<dose>   
│   │   │   │   ├── output_g1.pkl   
│   │   │   │   ├── output_g2.pkl   
│   │   │   │   ├── output_g3.pkl   
│   │   │   │   ├── output_g4.pkl   
│   │   │   │   └── output_g5.pkl
